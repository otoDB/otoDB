Welcome to the otoDB: the otomad/ytpmv database Beta. otoDB is a community-driven website focused on indexing the comprehensive body of work pertaining to Otomads (音MADs) and YouTube Poop music video (YTPMVs), consisting of a user-managed database and wiki. [Read more about otoDB](/post/2).

## Registration
Registration is now open to the public when provided an invite code. You can get an invite code from any Editor who already has an account. Editors may find the menu to create new invite codes in [Profile -> Edit].

## Database dumps
We have begun releasing database dumps, which may be useful for various purposes including but not limited to archiving and analytics. They are updated nightly [here](https://cdn.otodb.net/otodb-latest.sql.gz).
