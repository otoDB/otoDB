## What counts as a work?

What counts as an Otomad/YTPMV remains an unsettled topic. This website approaches this question pragmatically. Editors should mostly consider whether it is useful for the work to be indexed as an Otomad/YTPMV. In cases of dispute, the admin team has the final say. In particular:

- Medleys themselves should be registered as songs, not works.

## Which tag should be the base tag when performing a merge?

There is no singular answer. If the tags are in different languages:

- Tags in the Meta category should prioritize English as a base if it did not previously exist on an external site whose main language is not English.  
- Otherwise, prioritize the language used wherever the tag is most active (usually obvious for Source and Creator).

## What happens if a song or a creator is being sampled?

If the subject of a creator tag or a song tag is being sampled in a work, attach the tag and tick the special checkbox labelled “used as sample”. In other words, do not create tags such as `creator_name_(source)`.

## There is a self-referencing conflict with respect to the tag category.

1. If a creator makes a work which samples themselves: The authorship takes precedence, do not check the “used as sample” box. Use the separate tag <otodb-worktag slug="self_source" ></otodb-worktag>  .
2. Sampling a song to make a work that has the same song as a BGM: The BGM takes precedence, do not check the “used as sample” box. Use the separate tag <otodb-worktag slug="同曲madシリーズ" ></otodb-worktag>  .

## This work is using way too many sources\!

Use the `multisource` meta tag.
