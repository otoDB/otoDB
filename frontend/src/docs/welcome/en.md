### Welcome to the otomad/ytpmv database!

otoDB is a community-driven website focused on indexing the comprehensive body of work pertaining to Otomads (音MADs) and YouTube Poop music video (YTPMVs), consisting of a user-managed database and wiki.
Works are indexed from various sources, such as [YouTube](https://www.youtube.com/), [Niconico](https://www.nicovideo.jp/), [Bilibili](https://www.bilibili.com/), [SoundCloud](https://soundcloud.com/), and [Twitter](https://twitter.com/), enriched by user-contributed metadata, such as tags and relationships to other works.

otoDB also has a companion browser extension! This extension not only integrates with the website, but also adds convenience features for select websites, with more to come in the future. The current feature set consists of:

- Lookup if a work is indexed by otoDB
- View relevant work information, such as tags
- Quickly add a missing work to the otodb.net database
- For Niconico, we bypass geo-blocking -- **watch any video on Niconico, from any location; no VPN needed**! _(Requires a Niconico account.)_
